## requirements.txt

The file contains the environment specifications for the gaze-object-detection project

## marker.png

A picture of the aruco-marker attached on a ruler for training a model for gaze prediction. The marker can be any aruco-marker, but it has to be only one marker visible during the recording of data

## arucoBoard.pgn

A picture of the aruco-board used for extracting the real-world position of the detected objects. The aruco-board should be printed in its original dimensions.
